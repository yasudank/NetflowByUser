# PFS Netflow 観測準備パイプライン解説
### 観測者主体のファイバー配置設計・ファイル生成

すばる望遠鏡 PFS（Prime Focus Spectrograph）の観測準備における、視野ポインティングの最適決定から、ファイバーアサイン、観測ファイルの生成・検証までの自動統合プロセス。

---

# 1. 視野とガイド星の最適化
## `optimize_hex_fov_with_guidestars.py`

PFS の六角形視野（半径 $0.69^\circ$）内に最も多くの天体が収まり、かつガイドカメラに十分な星が入るポインティング $(RA, Dec, PA)$ を自動決定。

- **天体カバレッジ**: 候補点上で高速な六角形内外判定を実行。
- **ガイド星選定 (Auto Guide)**:
  - 6台のAGカメラ領域への投影。
  - $12.0 < G < 21.5$ 等の Gaia 星を抽出。近接星（$1.0\ \text{arcsec}$ 未満）は排除。
  - 視野内に輝星 ($G \le 12.0$) が入る場合はカメラ飽和のためペナルティ。
  - 最低 `min_cams_with_stars` 台で必要数確保。
- **故障コブラ回避**:
  - 故障ファイバーの位置に $12.0$ 等以下の輝星が来ないように制約（安全半径 $1.5\ \text{arcmin}$）。

---

# 1.2. 視野最適化の探索アルゴリズム
## Coarse-to-Fine 二段階探索

1. **粗い探索 (Coarse Search)**:
   - $0.03^\circ$ 刻みの候補点グリッド ＋ ターゲット自体の座標。
   - PA は $[0^\circ, 60^\circ)$ の範囲を $5.0^\circ$ 刻み。
   - カバレッジ順にソートし、上位候補の制約チェック。
   - 制約を完全に満たした時点で探索を**早期終了 (Short-circuit)**。
2. **細かい探索 (Fine Search / Local Refinement)**:
   - 最良候補の周囲に $0.002^\circ$ 刻みの微細グリッドを生成。
   - PA は $\text{PA} \pm 5.0^\circ$ の範囲を $0.5^\circ$ 刻みで探索。
   - 局所領域内でカバレッジと制約を再評価し、最終ポインティングを決定。

---

# 2. ファイバーアサイン最適化
## `netflow_solver.py`

決定したポインティングにおいて、各天体に対し約 2,400 本の光ファイバーを割り当てる最適化計算。Gurobi 数理計画ソルバーを用いて、整数線形計画問題（ネットワークフロー）として高速に解決。

- **天体優先度**: 科学天体の優先順位（Priority 0 が最優先）に従って重み付け。
- **物理的可動範囲**: 各ファイバー位置決め装置（Cobra）のパトロール領域（ドーナツ型）内のみアサイン可能。
- **衝突回避**: 隣接するファイバーの先端同士が衝突（距離 $< 2.0\ \text{mm}$）しないよう厳密に保護。
- **キャリブレーション・スカイファイバー数**: 装置キャリブレーションに必要な最低数を必ずアサイン。

---

# 3. 輝星退避アルゴリズム (Dummy Target Avoidance)
## `run_netflow.py`

割り当てが行われなかった正常なファイバー（Cobra）が、周囲の明るい星（$G \le 12.0$）の近くに放置されると、不要な強光が入り込みノイズの原因になります。

### ダミーターゲットによる退避プロセス
1. **検出**: 未割当コブラから $1.5\ \text{arcmin}$ 以内に輝星が存在するかチェック。
2. **極座標グリッドサーチ**:
   - コブラのパトロール領域内で、**輝星と真反対の方向**へファイバー先端を最大まで伸ばす物理座標を計算。
   - 隣接する他のファイバーと衝突しない条件（距離 $\ge 2.0\ \text{mm}$）を満たす中で、輝星から最も離れる点を算出。
3. **反映**: 退避位置を RA, Dec に逆算し、優先度 4 (`priority=4`) の「ダミーターゲット」としてアサイン。

---

# 4. 観測制御ファイルの自動生成・検証
## `make_pfs_design.py`

最終成果物をすばる望遠鏡およびPFSシステムが認識可能な標準フォーマットに変換。

- **`pfsDesign` FITSファイル**:
  - 各ポインティングでのファイバーID、天体名、RA, Dec、PFI物理座標などを記録。
  - 装置制御およびデータ解析パイプラインのインプット。
- **OPEファイル (Observation Procedure Execution)**:
  - 望遠鏡へのポインティング指示や露光時間等の観測コマンドスクリプト。
- **シミュレーション・検証**:
  - ファイバー同士の干渉や patrol region のチェックを行い、安全な観測を保証。
